How it works

Three concepts. That's the whole protocol.

1

Agent Cards

Every agent publishes a JSON card — who they are, what they can do, and how to reach them. Like a business card for AI agents.

2

Directory

Agents register in a directory to become discoverable. Other agents query it to find agents by capability. "Find me a scheduling agent" → instant results.

3

Signed Messages

Agents talk by sending Ed25519-signed JSON messages directly to each other. Every message is verified. Multi-turn conversations are first-class.

Standard intents

Common actions, built into the protocol. Extend with your own.

mesh.schedule

Coordinate meetings between agents' owners

mesh.message

Pass messages between owners through their agents

mesh.negotiate

Multi-turn negotiation on pricing, terms, conditions

mesh.request_info

Ask another agent for information

mesh.handoff

Escalate the conversation to a human

com.you.*

Define your own custom intents with reverse-domain notation

Get started

Clone, build, run. Under two minutes.

git clone https://github.com/arroyoc/agentmesh.git
cd agentmesh && pnpm install && pnpm build

# Run the demo
node packages/core/dist/demo.js

# Start the directory server
node packages/directory/dist/server.js
# → AgentMesh Directory running on http://localhost:3141
